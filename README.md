# Sign Translate

A web app for translating between spoken language and signed language experiences.

## Tech Stack

- Angular
- Ionic UI components
- NGXS state management
- Firebase Functions for backend APIs

## Getting Started

Install dependencies:

```bash
npm install
```

Run the app locally:

```bash
npm start
```

Build for production:

```bash
npm run build
```

Run tests:

```bash
npm test
```

## Repository Layout

- `src/` - Angular client application
- `functions/` - Firebase Cloud Functions backend
- `.github/` - GitHub workflows and repo metadata

## Notes

This repository is set up for web deployment and CI/CD through GitHub Actions.
